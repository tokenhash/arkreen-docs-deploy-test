Buy Remote Miner

Q1: What’s the difference between a Remote Miner and a Gaming Miner?​

A Gaming Miner is only designated for testing during the Phase 0 period of the Arkreen Network project. Hence, the electricity generation is only simulated and recorded on the testnet.

Whereas, a Remote Miner is based on a physical solar PV system, which unlocks tremendous opportunities for both web3 enthusiasts and those committed to embracing climate change issues head-on with real solutions. Arkreen Network provides a roadmap solution and a unique opportunity to those who cannot participate in direct rooftop solar PV panel installation but can still capitalize on renewable energy resources and remotely connect to the Arkreen Network. Interested renewable energy resource enthusiasts can join the Arkreen Network by purchasing a remote miner through the Arkreen Network dashboard Console Page. After purchasing the Remote Miner, a NFT is minted for each purchased Remote Miner.

Q2: Can I buy more than one Remote Miners?​

Yes. Unlike the Gaming Miner, there is no limit on the number of Remote Miners you can purchase to contribute towards a greener environment.

Q3: How can I buy a Remote Miner?​

The Remote Miner program is in the full testing phase right now, with 500 testing slots open to early-bird enthusiasts for free (until it actually launches, for detail, please follow us through Twitter/Discord/Telegram Group).

To purchase a remote miner, the following instructions will help ease users:

Preliminary Preparation​

Simply request a Gaming Miner and withdraw gAKRE to participate in the free Remote Miner Testnet Mining program. Start by withdrawing a certain amount of gAKRE needed to apply for and then buy a Remote Miner. The right amount of gAKRE required can be ascertained when requesting the Gaming Miner; a simple guide here helps explain how to apply for a Gaming Miner. If you have already set up a Gaming Miner and withdrawn gAKRE, you can skip this section.

1. Open Arkreen Console​

Launch your browser and visit Arkreen Network Console Page to log in with a MetaMask account to access arkreen’s dashboard console:

Users who log in for the first time must select the Polygon Mainnet and accept the license agreement according to the operational procedures documented.

2. Click Drop-down Menu​

A User can tap the icon at the top right corner of the page to view the dashboard console options.

3. Select "Buy a Remote Miner"​

Select to “Buy a Remote Miner” line item from the drop-down menu below.

4. Select available Remote Miner​

Select the available “RemoteMiner” in the line item drop-down menu screen, and select the green highlighted “Buy” line item from the Remote Miner list.

5. Buy and wait for a Remote Miner being allocated​

Please follow exact operational procedures and complete the purchasing process instructions; note that processing time may take a little longer. Your patience is appreciated!

6. Pay for the Remote Miner​

The payment transaction is processed by selecting the “Pay Now” button below, on the lower right side of the page.

Note: In the free Remote Miner testing process, the Remote Miner is paid with gAKRE, including the possibility of using other coins in the future.

7. Sign the purchase order​

The User must launch their MetaMask wallet to sign the purchase order.

8. Payment confirmation​

Proceed with the payment confirmation in the MetaMask wallet.

9. Wait for payment​

The Payment process is underway and may take several minutes to complete.

10. Transaction confirmation​

A payment indicator screen may appear (depending on your browser settings). Select the payment indicator screen above to view the transaction confirmation on the Polygonscan shown below:

Users can view the network transaction details from the Polygonscan below:

11. View your Remote Miner​

Upon completion of the Remote Miner purchase, you can select the Miners tab below to view the newly purchased Remote Miner while analyzing potential reward option details on the Arkreen Network.
